**CodeForge** is a modern, professional C++ Integrated Development Environment built with cutting-edge technologies. Designed for developers who demand performance, reliability, and an intuitive user experience.

## ✨ Features

### 🖥️ **Advanced Code Editor**
- **Syntax Highlighting** - Full C++ syntax support with customizable themes
- **Intelligent Line Numbers** - Dynamic line numbering with error indicators
- **Real-time Error Detection** - Instant feedback on compilation errors and warnings
- **Code Completion** - Smart auto-completion and suggestions
- **Multi-file Support** - Tabbed interface for multiple source files

### 🔧 **Powerful Build System**
- **Multi-compiler Support** - GCC, Clang, MSVC compatibility
- **Real-time Compilation** - Background compilation with live feedback
- **Custom Build Configurations** - Debug, Release, and custom build modes
- **Project Templates** - Pre-configured project structures
- **Dependency Management** - Integrated library and include path management

### 📁 **Project Management**
- **Workspace Organization** - Hierarchical project structure
- **File Explorer** - Integrated file browser with project navigation
- **Recent Projects** - Quick access to recently opened projects
- **Build System Integration** - CMake and Makefile support

### 🎯 **Developer Tools**
- **Output Console** - Detailed compilation output and error reporting
- **Performance Metrics** - Build time tracking and optimization suggestions
- **Cross-platform** - Native support for Linux, Windows, and macOS
- **Extensible Architecture** - Plugin-ready design for future enhancements

## 🚀 Quick Start

### Prerequisites
- **CMake** 3.16 or higher
- **C++ Compiler** (GCC 9+, Clang 10+, or MSVC 2019+)
- **GLFW3** development libraries
- **OpenGL** 3.3+ support

### Installation

1. **Clone the repository**
   ```bash
   cd /path/to/Megacraft3D
   cd builder
   ```

2. **Build CodeForge**
   ```bash
   chmod +x build.sh
   ./build.sh
   ```

3. **Run the application**
   ```bash
   cd build
   ./bin/CodeForge
   ```

## 🛠️ Usage

### Creating Your First Project
1. Launch CodeForge
2. Go to **File** → **New Project**
3. Choose a project template (Console Application, Library, etc.)
4. Select your preferred compiler settings
5. Start coding!

### Compilation and Building
- **Quick Compile**: Press `F7` or use **Build** → **Compile**
- **Full Build**: Press `Ctrl+F7` or use **Build** → **Build Project**
- **Run**: Press `F5` or use **Build** → **Run**

### Example Projects
CodeForge comes with built-in examples:
- **Calculator** - Demonstrates basic C++ concepts and STL usage
- **Game Engine** - Object-oriented programming with inheritance and polymorphism
- **Data Structures** - Common algorithms and data structure implementations

## 🏗️ Architecture

CodeForge is built using modern C++ and proven technologies:

- **Core Framework**: C++17 with STL
- **UI Framework**: Dear ImGui for responsive, immediate-mode GUI
- **Graphics**: OpenGL 3.3+ with GLFW for window management
- **Build System**: CMake for cross-platform compilation
- **JSON Configuration**: nlohmann/json for settings and project files
